## Keyboard shortcuts [OPEN]

### Overview

Currrenty, we don't have any keyboard shortcuts in our app. It would be great to have some keyboard shortcuts to improve user experience and make it easier to navigate the app. 

### Acceptance criteria

1. Play button can be triggered by pressing the spacebar.
2. Pause button can be triggered by pressing the spacebar.
3. Next button can be triggered by pressing the right arrow key.
4. Previous button can be triggered by pressing the left arrow key.
