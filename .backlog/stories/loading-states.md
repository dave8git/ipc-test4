## Loading states [OPEN]

### Overview

While loading files to the app we don't see any loader. It would be good to have some loading states to improve user experience and make it clear that the app is working on loading the files.

### Acceptance criteria

1. We have a loading state component that can be used to indicate that the app is loading files.
2. The loading state component is displayed when the app is loading files and hidden when the files are loaded.
3. The loading state component is visually appealing and consistent with the overall design of the app.
